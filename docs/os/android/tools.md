##### 通过电脑在google play的网站下载apk
- 第一步：打开google play官网：https://play.google.com/
- 第二步：打开下载APK的网址：https://apps.evozi.com/apk-downloader/
- 第三步：回到google play找到手机app应用，点击分享，然后复制链接
- 第四步：再回到下载APK的网址中把复制的连接复制到输入框中进行搜索
- 第五步：点击下载就可以了
