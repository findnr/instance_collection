### 时间戳转成时间格式
```sh
# 1.在表格选中一列空白设置成日期格式
# 2.选中此列中第一行，将公示输入公式： 
#   说明：如果时间戳是秒为单位的：=(E2+8*3600)/86400+70*365+19
#        如果时间戳是以毫秒为的单位的：=(E2/1000+8*3600)/86400+70*365+19
# 3.把E2改成相对应要转换的值的表格
# 4.选择中此列第一行，右下角下拉就可以了

```
